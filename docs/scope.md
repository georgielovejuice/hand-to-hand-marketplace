# Project Scope – Hand to Hand

## 1. Project Description
Hand to Hand is a web-based second-hand marketplace for students, staff, and lecturers at KMITL. 
The system allows users to post items, browse listings, and communicate with sellers through
a messaging and offer system.

## 2. Objectives
- Allow users to create, edit, and delete item listings.
- Enable browsing, searching, and filtering items.
- Provide buyer–seller messaging and offer negotiation.
- Allow admin moderation of users and items.

## 3. In Scope
- User authentication and profile management
- Item listing management
- Messaging between buyers and sellers
- Offer accept / reject / hold mechanism
- Admin moderation tools

## 4. Out of Scope
- Online payment systems
- Delivery or logistics services
- External marketplace integrations
- Rating or review systems

## 5. Constraints
- Semester timeline
- Limited team size
- Demo-based evaluation

## 6. Assumptions
- Users have internet access
- Users belong to KMITL community

## 7. Acceptance Criteria
- Users can register, login, and logout
- Items can be posted with images
- Messaging and offer system works correctly
- Admin can remove items or ban users
